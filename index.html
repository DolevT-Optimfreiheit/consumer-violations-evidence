Documentation package for consumer protection complaints (German/EU and Portuguese/EU).

Run the app

This project is a Streamlit app. GitHub Pages only serves static files, so the app does not run here.

  • Local: pip install streamlit && streamlit run app.py
  • Online: Deploy on Streamlit Community Cloud (connect this GitHub repo).
